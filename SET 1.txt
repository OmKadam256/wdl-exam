Q9 – File Upload Form with Checkboxes

Write a HTML code to generate following output.

Enter Name of your friend
Choose the file you want to post to your friend

Browse...

What does the file contain?
☑ Image   ☑ Source code   ☐ Binary code

You have Completed the Form . Submit Query

<!DOCTYPE html>
<html>
<body>
  <form>
    <table>
      <tr>
        <td>Enter Name of your friend</td>
        <td><input type="text" name="friendname" size="30"></td>
      </tr>
      <tr>
        <td colspan="2">Choose the file you want to post to your friend</td>
      </tr>
      <tr>
        <td colspan="2"><input type="file" name="uploadfile"></td>
      </tr>
      <tr>
        <td colspan="2">What does the file contain?</td>
      </tr>
      <tr>
        <td colspan="2">
          <input type="checkbox" name="type" value="image" checked> Image
          <input type="checkbox" name="type" value="source" checked> Source code
          <input type="checkbox" name="type" value="binary"> Binary code
        </td>
      </tr>
      <tr>
        <td>You have Completed the Form.</td>
        <td><input type="submit" value="Submit Query"></td>
      </tr>
    </table>
  </form>
</body>
</html>
